# GIT
## Principais comandos
- git init: Inicializar um repositório no Git
- git add *: Adicionar/Atualizar os arquivos 
- git commit -m "": Commitar os arquivos para poder empurrar no GitHub
- git remote add origin linkdorepositorio : Colocar os arquivos no repositório
- git push origin main: Empurrar o repositório para o GitHub
- git status: Saber a situação, se tem alguma coisa errada e como pode ser consertada
- ls: Listar os arquivos contidos
- ls a: Listar os arquivos ocultos contidos
- cd: Navegar entre os arquivos/diretórios
- git push -u origin master: Também empurrar os arquivos para o GitHub
- mkdir: Criar pastas
- git clone: Clonar um repositório
